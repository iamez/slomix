- **Zakaj ovrženo:** The code behavior is accurately described (proximity_helpers.py:184-186 builds `(guid = $n OR target_guid = $n)`, and proximity_teamplay.py:188 does `GROUP BY guid`), but the "bug" characterization rests on an unproven intent contract.
