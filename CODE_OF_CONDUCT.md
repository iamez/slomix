# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/) code of conduct.

## Our Standards

- Be respectful and constructive in technical discussion.
- Focus critique on ideas and implementation details, not people.
- Assume good intent and ask clarifying questions before escalating.

## Enforcement

Report conduct issues privately to the maintainers through a GitHub security advisory or direct maintainer contact listed in `SECURITY.md`.

Project maintainers are responsible for clarifying and enforcing this code of conduct.
