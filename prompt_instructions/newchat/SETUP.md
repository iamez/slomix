## Prerequisites
- Python 3.9+
- Discord Bot Token
- ET:Legacy Server

## Installation
1. Clone repository
2. Install requirements: `pip install -r requirements.txt`
3. Configure `.env` file
4. Run: `python bot/ultimate_bot.py`

## Configuration
Edit `.env` file with:
- DISCORD_BOT_TOKEN=your_token
- GUILD_ID=your_server_id
- STATS_CHANNEL_ID=your_channel_id
